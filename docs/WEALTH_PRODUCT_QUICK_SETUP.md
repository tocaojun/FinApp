# 银行理财产品快速设置指南

## 🚀 5分钟快速上手

### 第 1 步：打开资产管理

1. 登录 FinApp
2. 进入 **资产管理** 页面
3. 点击 **新增资产** 按钮

### 第 2 步：选择资产类型

在资产类型下拉列表中选择：**理财产品 (WEALTH)**

### 第 3 步：填写基本信息

| 字段 | 示例值 | 说明 |
|------|--------|------|
| 代码 | ICBC-2025-11 | 产品代码（唯一标识） |
| 名称 | 工商银行2025年11月理财产品 | 产品全名 |
| 市场 | 中国 | 选择 "中国" |
| 币种 | CNY | 选择 "人民币" |
| 风险等级 | 中等 | 对应产品的风险等级 |
| 流动性标签 | 中等 | 理财产品通常选 "中等" |

### 第 4 步：填写产品详情

在详情部分填写以下关键信息：

```
【基本信息】
  产品类型: 固定收益 (fixed_income)
  风险等级: R2 (对应中等风险)

【收益信息】
  预期收益率: 3.65 (%)
  最低收益率: 3.60 (%)
  最高收益率: 3.70 (%)
  收益类型: 年化收益 (annual)

【时间信息】
  发行日期: 2025-11-01
  起始日期: 2025-11-05 ← 投资开始计息日期
  到期日期: 2026-02-04 ← 重要！

【投资限制】
  最小投资额: 1000 (元)
  最大投资额: 1000000 (元)
  投资递增: 1 (元)

【发行方信息】
  发行机构: 中国工商银行
  产品代码: ICBC-2025-11-91

【赎回信息】
  支持提前赎回: 是/否
  赎回费率: 0.1 (%)
  锁定期: 5 (天)
```

### 第 5 步：保存产品

1. 点击 **保存** 按钮
2. 系统会创建该理财产品资产

## 💳 记录投资交易

### 步骤 1：进入交易记录

1. 进入 **交易记录** 页面
2. 点击 **新增交易** 按钮

### 步骤 2：填写交易信息

| 字段 | 示例 | 说明 |
|------|------|------|
| 交易类型 | 理财购入 | 选择 "理财购入" |
| 资产 | 工商银行2025年11月 | 选择刚才添加的产品 |
| 金额 | 50000 | 投资金额 |
| 单价 | 1 | 理财产品固定为1 |
| 交易日期 | 2025-11-05 | 实际购买日期 |
| 账户 | 交通银行 | 从哪个账户购入 |
| 标签 | 银行理财、固定收益 | 自定义标签便于分类 |
| 备注 | 91天期限产品 | 可选备注 |

### 步骤 3：保存交易

点击 **保存** 按钮

## 📊 查看投资情况

### 在投资组合中查看

进入 **投资组合** 页面，你可以看到：

- ✅ 理财产品的当前持仓金额
- ✅ 占投资组合的百分比
- ✅ 预期年收益金额
- ✅ 所有理财产品列表

### 查看具体产品信息

1. 进入 **资产管理**
2. 找到你的理财产品
3. 点击 **查看详情** 或 **编辑**
4. 可以看到完整的产品信息和收益计算

## 💰 收益计算示例

**假设场景：**
```
产品: 工商银行2025年11月理财
投资金额: 50,000 元
投资日期: 2025-11-05
到期日期: 2026-02-04
投资期限: 91 天
年化收益率: 3.65%
```

**系统会自动计算：**

```
年化收益 = 50,000 × 3.65% = 1,825 元/年

实际收益 = 50,000 × 3.65% × (91 ÷ 365)
        = 50,000 × 0.0365 × 0.2493
        = 455.27 元

到期本息合计 = 50,000 + 455.27 = 50,455.27 元
```

## 📅 到期处理

### 到期前 30 天

系统会自动提醒你：
- ✅ 产品即将到期（30天）
- ✅ 当前收益金额
- ✅ 需要决定是否续投

### 到期日

1. **资金自动回收**
   - 本金和收益自动转入现金账户

2. **记录收益交易**
   - 可在交易记录中添加 "理财赎回" 交易
   - 记录实际获得的收益

3. **更新投资组合**
   - 原理财产品变为已结束
   - 现金增加相应金额

## 🎯 常见操作

### Q: 如何修改已有产品的信息？

**A:** 
1. 进入 **资产管理**
2. 找到产品，点击 **编辑**
3. 修改信息后点击 **保存**

### Q: 如何删除错误的产品记录？

**A:**
1. 进入 **资产管理**
2. 找到产品，点击 **删除**
3. 系统会提示是否删除关联的交易记录

### Q: 产品提前赎回了怎么处理？

**A:**
1. 记录一条 "理财赎回" 交易
2. 金额填写实际获得的本息合计
3. 日期为实际赎回日期
4. 在备注中说明提前赎回和实际收益

### Q: 如何跟踪多个银行的理财产品？

**A:**
- 每个产品都是独立的资产
- 在 "标签" 中标注发行银行
- 可在投资组合中按发行银行筛选查看

## 📊 建议配置

### 初级用户（保守型）

```
推荐配置：
- 选择 R1-R2 低风险产品
- 选择 3-6 个月期限的产品
- 最小投资额通常为 1,000-5,000 元
- 保留 50% 资金在活期账户

示例：
- 50,000 元 × 60% = 30,000 元投理财
- 50,000 元 × 40% = 20,000 元留活期
```

### 中级用户（稳健型）

```
推荐配置：
- 配置 R2-R3 产品各占 50%
- 选择 3-12 个月期限的产品
- 根据现金流需求规划期限
- 保留 30% 资金在活期账户

示例：
- 100,000 元 × 30% = 30,000 元留活期
- 100,000 元 × 35% = 35,000 元投 R2 产品
- 100,000 元 × 35% = 35,000 元投 R3 产品
```

### 高级用户（积极型）

```
推荐配置：
- 配置 R2-R4 产品各占 1/3
- 选择 3-24 个月期限的产品
- 使用梯形配置错开到期日期
- 保留 20% 资金在活期账户

示例：
- 200,000 元 × 20% = 40,000 元留活期
- 200,000 元 × 27% = 54,000 元投 R2 产品
- 200,000 元 × 27% = 54,000 元投 R3 产品
- 200,000 元 × 26% = 52,000 元投 R4 产品
```

## ⚠️ 注意事项

1. **及时更新到期日期**
   - 确保到期日期准确，系统才能提醒
   - 避免错过赎回期限

2. **妥善保管产品合同**
   - 收益率、锁定期等以合同为准
   - 遇到纠纷时需要证明

3. **关注收益变化**
   - 部分浮动收益产品的收益可能变化
   - 定期更新预期收益信息

4. **税务申报**
   - 理财产品收益需要按规定申报个税
   - 建议咨询专业税务顾问

5. **风险提示**
   - R4-R5 产品可能本金浮亏
   - 需充分评估风险承受能力
   - 不要过度集中投资
