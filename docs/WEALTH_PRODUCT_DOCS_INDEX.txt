【核心概念速查】
================================================================================

余额型产品 (BALANCE)
文档位置：README → "核心差异对比"
简述：直接操作金额，无份额概念
典型产品：余额宝、银行定期存款

净值型产品 (QUANTITY)
文档位置：README → "核心差异对比"
简述：通过份额和净值操作，类似基金
典型产品：基金、股票

productMode 字段
文档位置：ANALYSIS_SUMMARY → "改进后的状态"
简述：用于区分两种产品类型的关键字段

快速方案 vs 完整方案
文档位置：QUICK_DECISION_GUIDE → "改进方案对比"
简述：快速方案花2-3天，完整方案花4天

表单切换逻辑
文档位置：CODE_EXAMPLE → "第3部分"
简述：根据 productMode 自动选择不同的表单组件
