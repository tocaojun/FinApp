   → 问题：配额限制 500/天

   → 问题：Tushare 仅支持中国市场
   ✅ 纠正：用 Yahoo Finance 处理全部

   → 问题：FRED 只有经济数据
   ✅ 纠正：用 Yahoo 或 EastMoney

❌ 错误5：同时用多个高成本源

   ✅ 纠正：组合使用免费源，必要时才付费

【马上开始】

【第一步】进入系统
  → 打开 http://localhost:3001
  → 登录系统

【第二步】创建同步任务
  → 转到 "数据同步"
  → 点击 "新建同步任务"
  → 选择数据源：Yahoo Finance（推荐）
  → 设置更新频率：每天
  → 保存并运行

【第三步】验证同步
  → 查看同步日志
  → 确认 total_records > 0 ✓
  → 数据库中有新记录 ✓
