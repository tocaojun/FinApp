📑 页面标签页
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

数据同步页面包含三个标签页：

1. 同步任务
   • 创建、编辑、删除任务
   • 立即运行任务
   • 支持三种调度方式：手动/定期/定时
   • 显示任务状态和执行结果

2. 数据源
   • 查看可用的数据源列表
   • 显示数据源的启用状态
   • 查看最后同步时间

3. 同步日志
   • 查看所有同步操作的历史记录
   • 显示执行状态、耗时、成功/失败数
   • 追踪数据同步过程

⚙️ 功能特性
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

任务调度方式：
   ✓ 手动      - 点击按钮立即执行
   ✓ 定期      - 每 X 分钟执行（如 60 分钟 = 每小时）
   ✓ 定时      - 使用 Cron 表达式定时执行

任务配置选项：
   ✓ 任务名称（必填）
   ✓ 任务描述（可选）
   ✓ 数据源（必填）
   ✓ 资产类型过滤（可选）
   ✓ 市场过滤（可选）
   ✓ 调度方式（必填）
   ✓ 间隔/Cron 表达式（条件必填）
   ✓ 回溯天数（必填）
   ✓ 覆盖已有数据（可选）
   ✓ 启用此任务（必填）

🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 登录应用
2. 点击左侧菜单 "系统管理 → 数据同步"
3. 点击 "新建任务" 按钮
4. 填写任务信息并提交
5. 点击播放按钮立即运行，或等待定时执行

常用 Cron 表达式：
   0 9 * * *     # 每天 9 点
   0 */4 * * *   # 每 4 小时
   0 0 * * 1     # 每周一 0 点
   0 0 1 * *     # 每月 1 号 0 点
