【��️ 系统架构】

前端层 (React + Ant Design)
  ├─ WealthProductDashboard
  │  ├─ 汇总卡片
  │  ├─ 产品列表
  │  ├─ 收益对比
  │  ├─ 趋势图表
  │  └─ 偏差分析
  └─ wealthApi 服务

中间层 (Express + TypeScript)
  ├─ WealthProductController
  ├─ wealth.ts 路由
  ├─ WealthProductReturnService
  └─ WealthMonitoringService

数据库层 (PostgreSQL)
  ├─ wealth_product_details
  ├─ wealth_product_transactions
  ├─ wealth_product_nav_history
  └─ wealth_product_alerts

【✨ 核心特性】

1️⃣ 双产品类型支持
   ├─ 分红型 (DIVIDEND)
   │  └─ 累计分红 vs 预期收益对比
   └─ 净值型 (NAV)
      └─ 当前价值 vs 投资成本对比

2️⃣ 自动化监控系统
   ├─ 定时扫描 (可配置Cron)
   ├─ 自动告警 (三级系统)
   ├─ 告警管理 (确认/解决)
   └─ 统计分析

3️⃣ 收益分析功能
   ├─ 偏差计算
   ├─ 趋势分析
   ├─ 原因诊断
   └─ 建议生成

4️⃣ 数据可视化
   ├─ 净值走势图 (30天)
   ├─ 日收益柱状图
   ├─ 偏差进度条
   └─ 统计图表
